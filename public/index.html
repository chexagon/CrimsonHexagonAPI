Crimson Hexagon API Demo About this Data

This Demo has been generated with data gathered from the Crimson Hexagon API. All visualizations have been constructed from
analyzed social media data stemming from the ForSight platform. This ranges from Facebook profile engagement metrics, to algorithmically
classified posts by positive, negative, and neutral sentiment. An accompanying GitHub repository has been created with instruction on
installing and building this app locally. For additional information on Crimson Hexagon and our software, please visit our website.
